# Activación de virtualización

Durante la instalación se detectó que la virtualización por hardware estaba deshabilitada.

Comprobación:

```bash
egrep -c '(vmx|svm)' /proc/cpuinfo
```

Resultado inicial:

```
0
```

En la BIOS se habilitó:

```
Intel Virtualization Technology
```

Tras reiniciar:

```bash
egrep -c '(vmx|svm)' /proc/cpuinfo
```

Resultado:

```
8
```

También se verificó:

```bash
lsmod | grep kvm
```

Resultado:

```
kvm_intel
kvm
```

La virtualización quedó correctamente habilitada.
